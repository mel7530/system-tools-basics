# Tastenkombinationen (macOS / Terminal / Editor)

## Universell
⌘A    Alles markieren  
⌘C    Kopieren  
⌘V    Einfügen  
⌘X    Ausschneiden  
⌘F    Suchen  
⌘S    Speichern  
⌘O    Öffnen  
⌘Q    Programm beenden  
⌘Z    Rückgängig  
⌘⇧Z   Wiederholen  

---

## Navigation & Fokus (macOS allgemein)
⌘Tab        Zwischen Programmen wechseln  
⌘`          Zwischen Fenstern derselben App  
⌘L          Cursor in Adress-/Eingabeleiste  
⌘W          Fenster/Tab schließen  
⌘N          Neues Fenster  
⌘⇧N         Neues Fenster 

---

## Terminal 
⌃A          Cursor an Zeilenanfang  
⌃E          Cursor an Zeilenende  
⌃U          Ganze Zeile löschen  
⌃K          Löschen ab Cursor bis Zeilenende  
⌃W          Wort links vom Cursor löschen  
⌃Y          Gelöschten Text wieder einfügen  
⌃C          Befehl abbrechen  
⌃D          Eingabe beenden / Shell verlassen  
⌃L          Bildschirm leeren (wie `clear`)  

---

## Terminal-Historie
↑ / ↓       Letzte Befehle durchgehen  
⌃R          Rückwärtssuche in der History (Tippen → Enter ausführen)

---

## VS Code / Editor-typisch
⌘P          Datei schnell öffnen  
⌘⇧P         Command Palette  
⌘/          Kommentar ein/aus  
⌘D          Nächstes gleiches Wort auswählen  
⌘⌥↑ / ↓     Zeile duplizieren  
⌘⌥L         Alles neu formatieren (kontextabhängig)

---

## Browser (Safari / Chrome)
⌘T          Neuer Tab  
⌘W          Tab schließen  
⌘⇧T         Geschlossenen Tab wieder öffnen  
⌘R          Seite neu laden  
⌘L          Adresszeile fokussieren  
⌘+ / ⌘-     Zoomen  
⌘0          Zoom zurücksetzen  

---

## Merksätze
- ⌘ = Programme / Oberfläche  
- ⌃ = Text & Terminalfluss  
- ⌥ = erweitern / variieren  
- ⌘L = fast immer „Eingabezeile fokussieren“
